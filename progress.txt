## What's built and working

**Backend** (`backend/`) — Node.js + Express + TypeScript + MongoDB
- JWT auth with roles (admin/user) — `src/routes/auth.routes.ts`
- Multi-tenant: every query filtered by `companyId` from JWT, never from user input
- Product CRUD — `src/routes/product.routes.ts`
- POST /api/chat — Gemini AI (`@google/genai`, model: `gemini-flash-latest`) with real tool calling against MongoDB
- Seed script: 2 companies (TechMart, FashionHub), 12 products each, 4 users
- TypeScript, zero compile errors

**Frontend** (`frontend/`) — React + TypeScript + Vite + Tailwind CSS
- Login / Register pages
- Dashboard: product grid, search, category filters, skeleton loading, admin CRUD modal
- Chat: bubble UI, typing indicator, quick-suggestion chips, error handling

**Running:**
- Backend: `cd backend && npm run dev` → http://localhost:3001
- Frontend: `cd frontend && npm run dev` → http://localhost:5173
- MongoDB: Atlas (connection string in backend/.env)
- Gemini API key: in backend/.env (GEMINI_API_KEY)
